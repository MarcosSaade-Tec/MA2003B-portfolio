# Case 03 — Cluster Analysis (MA2003B)

This folder contains the coursework for Case 03 from MA2003B Multivariate Analysis for Data Science.

Project summary
- Goal: Perform customer segmentation using clustering methods and interpret resulting groups.
- Files:
  - `retail_customer_data-1.csv` — dataset with customer attributes
  - `customer_clustering_analysis.ipynb` — notebook with preprocessing, clustering (k-means, hierarchical), and visualization
  - `instructions-1.pdf` — task instructions

How to open and run
1. Create a Python environment and install dependencies:

```bash
python -m pip install --upgrade pip
python -m pip install jupyterlab pandas numpy scikit-learn matplotlib seaborn
```

2. Open `customer_clustering_analysis.ipynb` and run the cells.
