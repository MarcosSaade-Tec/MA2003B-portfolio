# Case 01 — Factor Analysis (MA2003B)

This folder contains the coursework for Case 01 from MA2003B Multivariate Analysis for Data Science.

Project summary
- Goal: Explore and reduce dimensionality of customer satisfaction data using factor analysis methods.
- Files:
  - `customer_satisfaction_data.csv` — original dataset used for the analysis
  - `main.ipynb` — Jupyter notebook with data cleaning, exploratory analysis, and factor analysis
  - `Resumen_Ejecutivo.pdf` — short executive summary (Spanish)

How to open and run
1. Create a Python environment (Python 3.8+ recommended) and install Jupyter and common data packages:

```bash
python -m pip install --upgrade pip
python -m pip install jupyterlab pandas numpy matplotlib seaborn scikit-learn factor_analyzer
```

2. Start Jupyter and open `main.ipynb`.
