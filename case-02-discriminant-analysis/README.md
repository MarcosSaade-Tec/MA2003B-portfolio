# Case 02 — Discriminant Analysis (MA2003B)

This folder contains the coursework for Case 02 from MA2003B Multivariate Analysis for Data Science.

Project summary
- Goal: Build and evaluate discriminant analysis models for credit risk classification.
- Files:
  - `credit_risk_data-1.csv` — dataset used for classification
  - `LeadSmart_Analysis.ipynb` — Jupyter notebook with preprocessing, model training (LDA/QDA), and evaluation
  - `instructions.pdf` — instructor notes

How to open and run
1. Create a Python environment and install dependencies:

```bash
python -m pip install --upgrade pip
python -m pip install jupyterlab pandas numpy scikit-learn matplotlib seaborn
```

2. Open `LeadSmart_Analysis.ipynb` in Jupyter and run cells in order.
